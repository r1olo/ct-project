## Compiler Techniques Project

This repository hosts two custom compilers built for the Compiler Techniques course, part of the Master's Degree program at the University of Pisa.

The compilers were developed strictly for educational purposes and their architectural design was not heavily thought our or optimized, as the primary goal was simply to meet the requirements to pass the class. Using them for any real-life applications is not only discouraged, but also dumb.

## About the project

The project explores both imperative and functional programming paradigms by implementing basic compilers for two toy languages:

- *MiniImp*: an imperative programming language that links to a dummy C wrapper.
- *MiniFun*: a functional language that wanted to be OCaml so bad, but couldn't.
